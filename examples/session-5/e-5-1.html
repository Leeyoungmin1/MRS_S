CSS Position 속성 예제

relative, absolute, fixed, sticky 동작 비교

relative

position: relative는 요소의 원래 위치를 기준으로 이동합니다. 다른 요소의 레이아웃에는 영향을 주지 않으며, 이동 전의 공간은 그대로 유지됩니다.

relative

absolute

position: absolute는 가장 가까운 position 속성이 지정된 조상을 기준으로 배치됩니다. 기준 요소가 없으면 문서의 <html>을 기준으로 합니다.

absolute

fixed

position: fixed는 브라우저 뷰포트를 기준으로 고정됩니다. 페이지를 스크롤해도 항상 같은 위치에 유지됩니다.

sticky

position: sticky는 스크롤 위치에 따라 상대적으로 이동하다가, 지정한 오프셋(top 등)에 도달하면 해당 위치에 고정됩니다. 단, 속한 스크롤 컨테이너 영역을 벗어나면 더 이상 고정되지 않습니다.
